# Angular Content Projection

An Angular project demonstrating content projection patterns using `ng-content`, including single-slot, multi-slot, and conditional projection.

## Concepts Covered

- Single-slot content projection
- Multi-slot content projection with `select`
- Conditional content projection
- Server-side rendering configuration
- Angular routing with SSR support

## Tech Stack

- Angular 17+ (Standalone components)
- TypeScript
- Angular SSR

## Getting Started

### Prerequisites
- Node.js 18+
- Angular CLI

### Installation
```bash
npm install
ng serve
```

Visit `http://localhost:4200`
